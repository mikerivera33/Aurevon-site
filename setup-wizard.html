Create Collection

Create the Aurevon Genesis Drop collection

In the Crossmint Console, go to NFTs → Create Collection. Use these exact settings:

Collection Name:  Aurevon Genesis Drop
Blockchain:       Base (Base Mainnet)
Token Standard:   ERC-721
Supply:           5000 (or unlimited)
Royalty:          5% → your wallet address
Metadata Storage: Crossmint (managed)
Found in Console → Collection → Settings → Collection ID
