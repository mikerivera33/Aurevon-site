End-to-End Flow
1. Customer clicks Buy on RE Final or Web3 → unified payment terminal opens
2. Payment via Stripe Checkout (Card / Cash App / Afterpay) OR PayPal
3. On success: webhook hits /api/webhooks/stripe (or /api/webhooks/paypal)
4. Webhook → Airtable.createPayment() → row in Payments table
5. Webhook → Crossmint.mintToEmail() → NFT minted to customer email (no wallet)
6. Webhook → Airtable.createNftMint() → row in NFT_Mints table (status: Sent)
7. Webhook → Resend.sendNftDelivery() → branded email with NFT image + Discord invite
8. Customer clicks Discord invite link → /api/discord/auth (OAuth start)
9. Discord redirects to /api/discord/callback → role assigned by NFT type
10. Customer lands on /discord-welcome.html → tier-specific welcome shown
11. NFT_Mints row updated: Discord linked (Status → Minted)

Cron jobs:
- /api/cron/retry-mints (hourly): retries any Queued/Failed mints
- /api/discord/check-membership (daily): nudges customers who haven't joined Discord yet

▶ Webhook Setup Checklist
  • Stripe Webhook URL: /api/webhooks/stripe — listen for checkout.session.completed, payment_intent.succeeded
  • PayPal IPN URL: /api/webhooks/paypal — enable IPN in PayPal Business → Account Settings
  • Cash App for Business: Configure webhook in Square Dashboard (Cash App Pay uses Square's backend)
  • Afterpay: Set Afterpay Merchant Key in Stripe Dashboard → Payment Methods → Afterpay/Clearpay
  • Manual fallback: Export CSV from each processor monthly and import to Airtable Payments table
